# 1. Dasar Teori

Git adalah sistem yang digunakan untuk mengatur dan mencatat perubahan pada file dalam sebuah proyek perangkat lunak. Sistem ini dibuat oleh Linus Torvalds dan banyak dipakai oleh para pengembang karena dapat membantu melacak setiap perubahan kode yang dibuat. Dengan Git, pengguna bisa melihat riwayat perubahan, memperbaiki kesalahan, atau kembali ke versi sebelumnya jika diperlukan. Selain itu, Git menggunakan sistem terdistribusi, yang berarti setiap pengguna memiliki salinan repositori di komputernya sehingga pekerjaan tetap bisa dilakukan secara fleksibel. 

Dalam penggunaannya, Git memungkinkan pengguna untuk membuat repositori sebagai tempat penyimpanan proyek, menambahkan file, dan menyimpan perubahan yang telah dilakukan. Git juga dapat dihubungkan dengan layanan repositori online seperti GitHub agar proyek dapat disimpan dan dibagikan secara daring. Melalui fitur seperti commit, push, dan clone, Git memudahkan pengelolaan proyek sekaligus mendukung kerja sama antar pengembang dalam mengembangkan perangkat lunak.
